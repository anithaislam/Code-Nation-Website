1.5 Describing copyright and other constraints affecting websites

Copyright means the “right to copy” and it gives protection to the owners of material – artistic, musical, written or dramatic. Every country has different copyright laws. You need to know whether you have the permission to use the material you are including. [6] Other constraints that effect a website are technological. This could be browser compatibility or the resolution you use. [7] A website developer needs to ensure that their website appears correctly whether it is on a mobile phone screen or a laptop.

1.6 Describing access issues that need taking into account - describe issues that people with disabilities may face when using webpages and how we can overcome them

It is important that websites are designed in a way so that people with disabilities can use them. Unfortunately, this is not always the case therefore excluding people from accessing products and services. If you are blind/visually impaired, there needs to be “audio description of visual information”. This is information to accurately describe what can be seen on the screen. The website also needs to include extra time for those scenes which need additional description so the user has time to hear everything. As visually impaired people use screen readers (software application that interprets what is on the screen). It is important that websites are coded in a way so that the reader can correctly interpret the information. For example, each page should be given a unique title so that reader can discern it is a different page. [8]

1.7 Describe which are the appropriate filetypes for websites

It is important that you use the appropriate file types for your website. You can break the types into three – presentation, media and documents. For presentation, you could use either JPEG, GIF or PNG for images. HTML would be used for the web page and CSS to enhance the appearance of the website. For media – you have either MP3, MP4 or AVI which is video format. Finally ,the filetypes to use for documents would be doc, pdf or xls.
